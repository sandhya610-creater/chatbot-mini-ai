1 TITTLE:Development of a Simple AI Chatbot using Python

2.OBJECTIVE: The objective of this project is to design and implement a basic AI chatbot that can interact with users through text-based conversations. The chatbot responds to common queries such as greetings, jokes, time/date, and simple mathematical calculations.

3. INTRODUCTION: Artificial Intelligence chatbots are programs designed to simulate human conversation. This project demonstrates a simple rule-based chatbot built using Python. It uses pattern matching with regular expressions to understand user input and generate appropriate responses.
The chatbot is named Alex and is capable of handling multiple types of interactions in a conversational manner.

5. TECHNOLOGIES Used Programming Language: Python;Libraries Used random – for selecting random responses re (Regular Expressions) – for pattern matching datetime – for fetching current date and time
   
6. FEATURES: Interactive text-based conversation;Handles multiple types of queries;Provides real-time date and time;Tells jokes;Performs basic arithmetic calculations;Graceful exit system;Default responses for unknown inputs
   
7. WORKING:User starts the program;Chatbot greets the user;User enters a message
 
8. The chatbot:Converts input to lowercase;Matches it with predefined patterns;Returns a suitable response;Loop continues until user exits
   
9. Advantages
Simple and easy to understand
Lightweight and fast
No external dependencies
Good for beginners learning AI concepts

10. Limitations
Not a true AI (rule-based, not learning-based)
Limited understanding of complex queries
Cannot handle context-aware conversations
Weather responses are not real-time
