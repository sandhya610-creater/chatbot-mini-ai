OVERVIEW OF THE PROJECT :
The Simple AI Chatbot is a Python-based program that simulates basic human conversation using predefined rules and pattern matching.
It can respond to greetings, answer simple questions, tell jokes, display the current date and time, and perform basic math calculations.
The chatbot works through a command-line interface and demonstrates fundamental concepts of chatbot design in a simple and beginner-friendly way.
